# Security Policy

## Scope

This policy covers `kekse` (the published cookie codec) and `rfc_6265`.
`keksbruch` is an internal differential test harness and is not intended for production use.

## No warranty

This is free, non-commercial open-source software provided without warranty (see [`LICENSE`](LICENSE)).
Reporting a vulnerability creates no obligation, and no guarantee of a fix or of any timeline.

## Reporting a vulnerability

Please report suspected vulnerabilities **privately** via GitHub Security Advisories:
<https://github.com/gronke/kekse/security/advisories/new>

Please do not open a public issue for an undisclosed vulnerability.

## Supported versions

Only the latest `0.x` release is maintained.
As a `0.x` project, APIs and security behaviour may change between releases.
